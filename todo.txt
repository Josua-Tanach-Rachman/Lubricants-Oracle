00A - 00B Welcome Popup

The popup should be displayed if user accesses the website the first time
01A - 01C Home Page

User can either do free text search or browser by category / application
If do free text search, the search popup from 02 is displayed
User can tap the product category / application to select a value, and then the "Browser X Products" is updated with the count of the matched products. Then user can click the Browser button to go to the Product Search Results Page
The promotions, most searched products and links are from backend too
02A - 02C Search Popup

When user types in the search box, the list should be updated to show the matched countries, product families, ports, product ranges, products, product categories, and applications.
User can click See-All in search results to go to the corresponding search results page
User can tap a specific entity in search results to go to the details page
The product-categories and applications are also listed in Browser section (and user can scroll down to load more)
User can tap the parent product category / application to show the child list
User can tap the specific child and go to the products search result page
03A - 03F Product Search Results Page

The matched products can be grouped by country, port, family, range, or not grouped (product tab)
User can filter the matched products further more
04A - 04B Product Family Details Page

The products belonging ranges are grouped by range, and the products not belonging to ranges are in the family group
User can filter the products further more
05 Product Range Details Page

The product range details page lists all the products in this range
06 Product Details Page

The product details page shows the product details.
04C - 04D Product Comparison

user can compare products on any page where the products can be selected (product search results page. family page, range page)
the smaller comparison floater shows when user checks any product, and disappears if no product is checked
07, 08 Contact

The contact popup is displayed wherever the Contact Us button is clicked (family, range, and product)
